## Setup
```shell
# Seleciona a versão compatível do Node
$ nvm use
# Instala as dependências
$ npm i
# Copia arquivo contendo variáveis de ambiente
$ cp .env.dev .env
# Roda api mock
$ node src/mock/api-mock/server.js
# roda o projeto
$ np run dev
```
